# Git Scanning Infrastructure

## Структура на репозиториумот

Репозиториумот е составен од AWS CDK (Cloud Development Kit) инфраструктура за автоматизирано скенирање на Git репозиториуми. [main.ts](iac/src/main.ts) е главниот entrypoint за CDK апликацијата каде што се креира CDK App објектот и се инстанцира стекот. [stack.ts](iac/src/stack.ts) дефинира `GitScanningStack` класа која е основниот градбен блок на CDK апликацијата и ја содржи целата инфраструктурна дефиниција.

Стекот ја прикачува целата инфраструктура која вклучува: VPC мрежа, ECR репозиториум за Docker слики, S3 кофа за извештаи од скенирањето, Secrets Manager за GitHub токен, AWS Batch окружување (Fargate Compute Environment, Job Queue и Job Definition) за извршување на задачите, како и EventBridge правило за автоматско активирање секоја недела. Безбедноста е зајакната преку строги bucket policies - S3 кофата може да биде пристапена **само** од ролјата на Batch задачите и од специфичниот корисник `filipjosifoski`, додека сите останати пристапи се блокирани, а комуникацијата е енкриптирана со SSL.
